# Pre-Delivery Checklist

## Visual Fidelity

- [ ] Typography hierarchy matches source intent.
- [ ] Core color palette matches source tokens.
- [ ] Spacing and alignment are consistent with source structure.
- [ ] Radius and shadow levels match source components.
- [ ] No layout shift caused by hover/focus states.

## Structural Fidelity

- [ ] Semantic regions are preserved.
- [ ] Repeated blocks are consistently implemented.
- [ ] Scope boundary is respected (page/component/section).

## Interaction Fidelity

- [ ] Buttons, links, and form controls preserve semantics.
- [ ] Keyboard focus visibility exists for interactive controls.
- [ ] Form labels and associations are preserved.

## Accessibility Baseline

- [ ] Images include meaningful alt text where needed.
- [ ] Headings follow logical order.
- [ ] Interactive elements remain reachable and operable.

## Delivery Integrity

- [ ] Changed files are explicitly listed.
- [ ] Known deviations are graded P0-P3.
- [ ] Risks and assumptions are documented.
